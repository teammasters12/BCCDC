What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page semantically and is the foundation of all websites.

HTML uses a series of elements (represented by tags) to define different parts of content such as headings, paragraphs, links, images, etc. Web browsers read HTML documents and render them into visible or audible web pages.

Note: HTML is not a programming language; it's a markup language that tells browsers how to structure the web content.

Why Learn HTML?

Learning HTML gives you the power to:

  • Create your own websites from scratch
  • Understand how web pages work
  • Customize existing templates
  • Communicate effectively with web developers
  • Lay the foundation for learning CSS and JavaScript

HTML Document Structure

Every HTML document follows a basic structure that includes several key components:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My First HTML Page</title>
</head>
<body>
  <h1>Hello, World!</h1>
  <p>This is my first web page.</p>
</body>
</html>

Let's Break Down These Elements:

  • <!DOCTYPE html> - Declares the document type and HTML version
  • <html> - The root element that contains all other HTML elements
  • <head> - Contains meta-information about the document
  • <meta> - Provides metadata such as character set, viewport settings
  • <title> - Specifies the title shown in the browser tab
  • <body> - Contains all the content that is visible on the page
  • <h1> - Defines a main heading
  • <p> - Defines a paragraph
Remember: HTML elements typically have an opening tag, content, and a closing tag. Some elements (like <meta>) are self-closing and don't need a closing tag.

Links

Links allow users to navigate between web pages. They are created using the <a> (anchor) element:

<!-- Basic link -->
<a href="https://www.example.com">Visit Example.com</a>

<!-- Link to another page in the same website -->
<a href="about.html">About Us</a>

<!-- Link that opens in a new tab -->
<a href="https://www.example.com" target="_blank">Open in New Tab</a>

<!-- Link to a specific section on the same page -->
<a href="#section-id">Jump to Section</a>

Images

Images are added to web pages using the <img> element:

<!-- Basic image -->
<img src="image.jpg" alt="Description of the image">

<!-- Image with width and height attributes -->
<img src="logo.png" alt="Company Logo" width="200" height="100">

<!-- Image with a link -->
<a href="https://www.example.com">
  <img src="banner.jpg" alt="Click to visit our website">
</a>
Important: Always include the alt attribute for accessibility. It provides alternative information for users who cannot view the image and helps with SEO.

Lists

HTML provides three types of lists:

Unordered Lists

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
  • Item 1
  • Item 2
  • Item 3

Ordered Lists

<ol>
  <li>First step</li>
  <li>Second step</li>
  <li>Third step</li>
</ol>
  1. First step
  2. Second step
  3. Third step

Definition Lists

<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>

Tables

Tables are used to display data in rows and columns:

<table>
  <thead>
    <tr>
      <th>Name</th>
      <th>Age</th>
      <th>Country</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>John Doe</td>
      <td>25</td>
      <td>USA</td>
    </tr>
    <tr>
      <td>Jane Smith</td>
      <td>30</td>
      <td>Canada</td>
    </tr>
  </tbody>
</table>
